#### 消除大数据认识的误区

- 大数据不等于hadoop
- 大数据不等于社交媒体数据（互联网数据等等）
- 大数据不等于技术（大数据包含技术和思维）
- 大数据不等于分析（大数据可能还跟分析、存储、计算有关系）
- 大数据不等于一种解决方案
